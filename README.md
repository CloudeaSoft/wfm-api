# wfm-api

Community TypeScript/JavaScript wrapper for the [Warframe Market](https://warframe.market) HTTP API.

**Unofficial.** Not affiliated with Warframe Market / 42bytes.

## Install

```bash
pnpm add wfm-api
# or
npm install wfm-api
```

## Status

Extracted from [`koishi-plugin-warframe`](https://github.com/CloudeaSoft/koishi-plugin-warframe). API surface and packaging are still evolving.
